## 搬瓦工的中国优化线路到底分几层

很多人搞不清搬瓦工一长串术语，其实拆开就三条主线：

**电信方向：CN2 GIA / CN2 GIA-E / CTGNet**
CN2 GIA（AS4809）是电信最贵、最稳的线路；CTGNet（AS23764）是电信新加的，搬瓦工官方说实际表现等同于 CN2 GIA。CN2 GIA-E（E-Commerce）本质就是 CN2 GIA 的电商版套餐，区别在于带宽从 2.5Gbps 起步、可选机房更多、可自由迁移。

**移动方向：CMIN2（AS58807）**
中国移动的高端直连线路，搬瓦工在洛杉矶 USCA_9 机房走的就是这条。

**联通方向：China Unicom Premium（AS10099，俗称 CUP / 9929）**
联通面向高端用户的国际优化线路。搬瓦工的 USCA_9 数据中心明确把 CN2 GIA + CMIN2 + CUP 三条线一起 outbound 给中国方向，这是目前它"三网优化"的标配组合。

## 不同需求场景下，到底怎么选

光看价格表是选不出来的，得对号入座。下面把几个最常见的真实需求拆开讲。

**场景一：个人建站 / 博客 / 小型外贸站**

预算有限、流量不大、希望国内访客也能流畅打开。直接看 CN2 GIA-E 入门款（$49.99/季 或 $169.99/年）。2.5Gbps 带宽对个人站完全过剩，1TB 月流量也跑不满，机房可以在 DC6、DC9、JPOS_1 等十几机房里随时切换测试，找到对你所在地最快的那一个固定下来。

如果你预算更紧，又对延迟不敏感（比如站主要给海外用户看），可以考虑 KVM 入门 $49.99/年 这一款，但要心里有数：它走普通线路，国内晚高峰会有感知。

**场景二：跨境业务 / 自建节点 / 中转**

对稳定性、晚高峰表现有硬要求。这时直接跳到 CN2 GIA-E 进阶款（$89.99/季 或 $299.99/年，2 核 3 线程 / 2G / 2TB / 2.5Gbps）。理由很简单：1GB 内存跑代理中转会比较吃紧，2GB 才有余量放各种服务；2TB 流量也比 1TB 更不容易触顶。

如果带宽需求更高，再往上的 4 核 4G（$56.99/月）开始带宽依然 2.5Gbps，到 6 核 8G 才升级到 5Gbps。这是搬瓦工故意做的阶梯——它知道大多数跨境用户其实吃不满 2.5Gbps。

**场景三：低延迟、电信用户为主**

如果你或你的用户主要是电信宽带，对延迟有洁癖（比如做远程桌面、跑游戏代理、ERP 远程办公），香港 CN2 GIA 是物理距离最近的选择。电信方向延迟通常在 30-50ms 区间，晚高峰也基本不抖。代价是价格——入门款 $89.99/月，是同规格 GIA-E（洛杉矶）的差不多 4 倍。

如果嫌香港贵，东京 CN2 GIA 是折中方案，三网回程都是 CN2 GIA，$89.99/月起步，对电信和联通用户都很友好；大阪 CN2 GIA 同样是日本节点，价格直接砍半（$49.99/月 起），带宽还更高（1.5Gbps vs 1.2Gbps），预算敏感的可以优先看大阪。

**场景四：移动用户为主**

中国移动用户去洛杉矶 GIA-E 系列机房反而体验更好，因为搬瓦工 USCA_9 走的是 CMIN2 直连，回程专门给移动优化过。新加坡 CN2 GIA 也是移动用户的热门选择，物理路径更近。

**场景五：高可用电商业务**

如果你跑的是不能宕机的电商业务、有 SLA 合规要求，那 CN2 GIA-E 已经不够了，要看电商 SLA 系列。99.99% 在线率保证 + 每 2 周免费换一次 IP，这两个功能对国内 IP 被封禁频繁的业务来说是刚需。
